ANCHOR TAG

Anchor tag o etiqueta de ancla representada por una letra a, para enlaces URL, que pueden estar contenidos dentro de nuestro archivo o en URL externas. Y sus atributos.

DOWNLOAD

Indica que el enlace se descargara en un archivo, en lugar de abrirlo, en algunos casos si lo abre si el navegador considera seguro abrirlo para una vista previa. Funciona cuando href hace referencia a un archivo que puede ser PDF, WORD, PGN, JPEG, ZIP, entre otros. Se puede forzar su descarga con el nombre original o cambiarlo agregando un valor al atributo.

Download image

Download image

HREF

Del ingles hypertext reference, signfica referencia de hipertexo, le indica al navegador donde abrir el enlace creado con la etiqueta a. Si no se coloca href el enlace no funciona, solo tendra apariencia de enlace pero no llevara a ningun lado.
