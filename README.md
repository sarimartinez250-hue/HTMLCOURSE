# 🌐Curso de HTML5

Proyecto de aprendizaje enfocado en el uso de etiquetas de HTML5, basado en la documentación de MDN.

## 📌Contenido
- Uso de la etiqueta `<a>` (anchor), atributos y ejemplos
- Uso de la etiqueta `<img>`, atributos y ejemplos
- Implementación de mapas de imagen (images map)
- Representación de regiones de Brasil mediante coordenadas
- Descripción de las regiones de Brasil

## ⚙️Tecnlogías
- HTML5
- PHP (procesamiento de coordenadas con atributo `ismap`)

## 🧠Descripción

Este proyecto fue creado como práctica para comprender en profundidad el uso de etiquetas HTML, apoyándome en la documentación de MDN.

Se exploraron atributos modernos y obsoletos, especialmente en la etiqueta `<img>`. Para entender el funcionamiento del atributo `ismap`, se implemento lógica en PHP para procesar coordenadas.

Posteriormente, se trabajó con mapas de imagen utilizando atributos como `map` y `coords`, aplicando estos conceptos en un ejemplo práctico basado en el mapa de Brasil y sus regiones.
